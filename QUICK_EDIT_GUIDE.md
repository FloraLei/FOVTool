# 车道线快速编辑和导出选项 - 使用指南

## 新增功能概述

### 1. 车道线快速编辑面板 ✅

左侧"车道线列表"面板现在包含快速编辑功能：

#### 快速编辑属性
选择车道线后，可以直接在下方的属性面板中修改：
- **名称** - 车道线标识
- **位置X** - 横向位置（-20 到 20 米）
- **起点Y** - 纵向起点（-500 到 500 米）  
- **终点Y** - 纵向终点（-500 到 500 米）
- **曲率** - 曲线参数（-0.1 到 0.1）
  - 0 = 直线
  - 正数 = 向右弯曲
  - 负数 = 向左弯曲
- **宽度** - 线条宽度（0.01 到 1.0 米）
- **颜色** - 颜色代码（如 #FFFFFF 或 FFFFFF）

#### 快速建立车道线流程
1. 选择车道线类型（直线/曲线/汇入/汇出）
2. 点击"+ 添加"按钮
3. 输入名称
4. 使用属性面板快速调整宽度、曲率等参数
5. 修改会实时更新到2D画布

### 2. 导出时选择是否包含车道线 ✅

在导出对话框中新增选项：

#### 导出 PNG 选项
- **渲染模式** - 选择高质量(推荐)或标准模式
- **分辨率** - 导出宽度
- **旋转** - 导出时旋转角度
- **背景颜色** - 白色/透明/黑色等
- **图例** - 是否显示传感器图例（左侧）
- **传感器过滤** ✅ NEW - 仅在图例中显示启用的传感器
- **车道线** ✅ NEW - 是否导出车道线（默认勾选）
- **抬头** - 导出图标题

#### 导出流程
1. 点击菜单 "📄 导出 PNG"
2. 选择"车道线"复选框（默认已勾选）
3. 配置其他导出选项
4. 点击"导出"
5. 选择保存位置

#### 导出选项影响
- 如果勾选"车道线" → 导出的图像中显示所有启用的车道线
- 如果取消勾选"车道线" → 导出的图像不包含车道线
- 两种模式都支持：高质量Matplotlib导出和标准Canvas导出

### 3. 高效创建车道线的技巧

#### 方法A：手动设置（精确控制）
```
1. 添加车道线 → 选择类型
2. 编辑属性：
   - 位置X: -1.0 (左侧1米)
   - 起点Y: -50.0 (车后50米)
   - 终点Y: 50.0 (车前50米)
   - 曲率: 0.0 (直线)
   - 宽度: 0.15 (标准线宽)
```

#### 方法B：快速复制（批量创建）
```
1. 创建一条标准车道线
2. 选中后点击"⧉ 复制"
3. 修改副本的"位置X"属性
4. 重复上述步骤创建多条平行线
```

#### 方法C：一键创建复杂场景
```
# 创建标准双向四车道
1. 左分界线：x=-2.0, type=straight
2. 左内线：x=-1.0, type=straight
3. 右内线：x=1.0, type=straight
4. 右分界线：x=2.0, type=straight

# 创建弯曲道路
1. 曲线左线：x=-1.0, curvature=0.01
2. 曲线右线：x=1.0, curvature=0.01
```

### 4. 属性值参考表

| 属性 | 范围 | 默认值 | 含义 |
|------|------|--------|------|
| 位置X | -20 ~ 20 m | 0 | 横向位置（负=左，正=右） |
| 起点Y | -500 ~ 500 m | -50 | 纵向起点（负=后，正=前） |
| 终点Y | -500 ~ 500 m | 50 | 纵向终点 |
| 曲率 | -0.1 ~ 0.1 | 0 | 曲线弧度系数 |
| 宽度 | 0.01 ~ 1.0 m | 0.15 | 线条宽度 |

车道线配置自动保存到JSON文件：
```json

    {

      "y_end": 50.0,
      "curvature": 0.0,

      "color": "#FFFFFF",
      "enabled": true
```

## 快捷键和技巧

- **快速选择** - 点击列表中的车道线名称
- **快速编辑** - 选中后在下方属性框修改
- **快速复制** - 创建相似车道线的最快方式
- **快速颜色** - 输入颜色代码时无需#号开头
- **快速禁用** - 点击"◎ 切换"显示/隐藏而不删除

## 导出建议

### 推荐导出配置
- **模式**: 高质量 (Matplotlib)
- **分辨率**: 4000px （出版质量）
- **背景**: 白色（打印友好）
- **图例**: ✓ （显示）
- **车道线**: ✓ （显示）
- **抬头**: 可选（添加标题增加专业感）

### 性能提示
- 直线车道线 → 渲染最快
- 曲线车道线 → 计算20+点，适度快速
- 复杂场景 → 可临时禁用不必要的车道线再导出
